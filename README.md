# ember-cli-intercom

Easily add interakt to your app.

## Installation

```bash
# From within your ember-cli project
ember install ember-cli-interakt
```

## Usage

First you'll need to configure your Intercom App ID:

```javascript
// config/environment.js

ENV['ember-cli-interakt'] = {
  appId: '[YOUR_APP_ID]'
};
```
